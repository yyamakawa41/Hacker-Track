Week 1 Interview Questions

1) What is the difference between HTML and XTML? Insert answer: - EXtensible HyperText Markup Language - Strict HTML for less leeway in coding; HTML + XML - XHTML is strict so to allow smaller devices (ie: tablets & smartphones) with less computing power to receive “proper” HTML coding
2) Why is it generally a good idea to position CSS links between HTML head tags and JavaScript script tags just before the html body tag? Do you know any exceptions? Insert answer: - It is for a better user experience; so that the viewer is seeing a completed version of the page when it finishes rendering instead of waiting while the page is loading. - HTML4 specifies that is to be coded between
3) What is the difference between classes and ID's in CSS? Insert answer: - ID: can only be invoked once, specify one specific tag/DIV - Class: used to encompass a bigger group; categorize
4) Describe Floats and how they work. Insert answer: - decimal number with digits on either side of the decimal point - fractions
5) What's the difference between a relative, fixed, absolute, and statically positioned element? Insert answer: - Statically: the default position, where it normally is - Relative: setting it so that the position is relative to where it normally is - Fixed, fixed position on a page, even if you scroll, it will not move - Absolute: it is setting the location exactly where you want it on the page, in relative to the bigger div
